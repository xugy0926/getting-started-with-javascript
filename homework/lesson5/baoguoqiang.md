##  *作业
 
 >function output(name) {
     console.log(name);
 }

 >var name = 'Hello,JS';
 
 >output(name);

关于上面几个"name"变量的差异：
1. function output(name)中的name是函数名下的一个参数变量；
2. console.log(name)中的name是函数内部中的一个局部变量；对于函数的内部调用有着至关重要的作用
3. var name 中的name是整个函数外部的全局变量；
4. 最后一个是调用函数所需的变量。
