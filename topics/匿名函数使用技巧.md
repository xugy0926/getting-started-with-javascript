以下例子是一个模拟异步处理的手法

```
// 定义一个函数，函数名为callback
function callback () {
  console.log('我终于被调用了');
}

// 5秒钟后调用callback函数
setTimeout(callback, 5000);
```

流程注解：
1. 定义了一个函数callback。
2. 然后调用setTimeout函数，5秒钟后调用callback函数。


其实，我们可以采用匿名函数来简化流程。

```
setTimeout(function () {
  console.log('我终于被调用了');
  }, 5000);
```

通过匿名函数的方式带来直接的好处就是，不用去考虑怎么定义函数的命名。提高了写代码的效率。
